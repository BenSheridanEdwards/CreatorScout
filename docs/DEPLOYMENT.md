Creator Scout can be deployed locally or on a VPS.

## Deployment Options

| Option | Best For | Docs |
|--------|----------|------|
| **VPS** | Production, full automation | [VPS_DEPLOYMENT.md](VPS_DEPLOYMENT.md) |
| **Local** | Development, testing | See README Quick Start |

## Prerequisites

- Node.js 18+
- PostgreSQL (Supabase/Neon free tier or self-hosted)
- AdsPower (browser fingerprinting)
- SmartProxy or similar (residential proxies)
- OpenRouter API key (vision AI)
